Repo Sleep Doctor
Release Risk Scanner for Repositories

上线前，不再“靠感觉”发版

Repo Sleep Doctor 会在发布前自动扫描仓库风险：冲突标记、潜在密钥、调试残留、README/脚本/测试缺失，并输出可读报告与 SARIF。

查看 GitHub 仓库 查看文档 
npm install
node src/cli.js . --format html --out reports/scan.html --fail-on p1

核心能力

发布阻断识别

从 P0/P1/P2 分级输出风险，帮助团队先处理真正影响上线的问题。

基线增量治理

支持 baseline + only-new，允许历史债务逐步清理，不阻断当前交付节奏。

平台接入友好

同时支持 text/json/markdown/sarif/html，便于接入 PR 检查与可视化平台。
