一键演示命令

可直接复制到 CI 或本地演示

# 1) 发现并导出仓库清单
node src/cli.js fleet-scan --discover-root ../workspace --discover-depth 4 --export-repos reports/repos.txt --export-only

# 2) 批量扫描并输出 Fleet 看板
node src/cli.js fleet-scan --repos-file reports/repos.txt --history-dir reports/fleet-history --scan-out-dir reports/fleet-reports --scan-format html --format html --out reports/fleet.html --continue-on-error --fail-on p1

# 3) 用 site/ 做宣传页发布目录
npx serve site
