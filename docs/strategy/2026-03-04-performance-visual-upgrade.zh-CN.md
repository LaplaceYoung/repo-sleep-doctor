# 竞品导向升级记录：可视化与性能（2026-03-04）

## 本轮目标

1. 提升报告可视化能力，增强“可读、可分享、可治理”。
2. 提升扫描运行速度，降低在中大型仓库的 CPU 开销。

## 已落地

1. HTML 报告升级为交互式仪表盘：
   - 严重级别占比环图
   - Top Rules 横向分布
   - Hotspot Files 表格
   - Findings 搜索与按严重级别过滤
2. 扫描引擎加入快速路径启发式：
   - 通过关键词预判是否需要逐行解析
   - 对低信号文本文件跳过逐行循环
   - 在报告中输出分析统计（textCandidates / lineScanSkipped / linesScanned）
3. 禁用规则执行优化：
   - 对 `preset` 或 `disabledRules` 关闭的规则，直接跳过匹配计算
   - 降低 release/security 专项扫描下的无效正则开销

## 竞争力提升点

1. 可视化从“静态表格”升级为“治理面板”，更接近头部 DevSecOps 工具体验。
2. 性能从“全量逐行”升级为“启发式分层扫描”，对大型仓库更友好。
3. 输出层新增性能可观测指标，有利于后续持续优化和对外展示。
